integrity.walkers documentation

lookup

fn

[input-schema reference-type lookup-ref]

Returns a function that takes a single object data, and returns a new object of the same shape with any references replaced with the result of calling (lookup-ref data)

To determine whether some part of the data is a reference, it checks the corresponding section of input-schema for equality with reference-type
